# Tremplin Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Construire un site de coaching prépa avec sidebar fixe, 3 pages service et booking Calendly.

**Architecture:** Next.js App Router avec un layout global contenant la sidebar fixe. Chaque service est une route (`/lettre`, `/cv`, `/oraux`). Le contenu de chaque service est centralisé dans `lib/services.ts`. Pas de backend.

**Tech Stack:** Next.js 15 (App Router), TypeScript, Tailwind CSS 4, 21st.dev Magic MCP pour composants UI, Calendly popup widget (CDN), Vercel pour le déploiement.

---

## Structure des fichiers

```
app/
  layout.tsx           → Layout racine : sidebar + slot main content
  page.tsx             → Redirect vers /oraux
  globals.css          → Variables CSS, reset
  oraux/page.tsx       → Page service Oraux
  lettre/page.tsx      → Page service Lettre de motiv
  cv/page.tsx          → Page service CV
components/
  Sidebar.tsx          → Navigation sidebar fixe (desktop) + toggle mobile
  ServicePage.tsx      → Template générique d'une page service
  Hero.tsx             → Hero card avec gradient indigo
  Stats.tsx            → Ligne de 3 stats
  Steps.tsx            → Liste d'étapes numérotées
  CTA.tsx              → Bouton Calendly + bouton contact
lib/
  services.ts          → Données des 3 services (contenu, liens Calendly)
  types.ts             → Types partagés (Service, Step, Stat)
__tests__/
  Sidebar.test.tsx
  ServicePage.test.tsx
```

```typescript
        </button>
      </div>
      {open && (
        <nav className="border-t border-gray-800 pb-2">
          {serviceOrder.map((slug) => (
            <Link
              key={slug}
              href={`/${slug}`}
              onClick={() => setOpen(false)}
              className={`flex items-center gap-3 px-5 py-3 text-sm ${
                pathname === `/${slug}` ? 'text-white bg-gray-800' : 'text-gray-400'
              }`}
            >
              <span>{icons[slug]}</span>
              <span>{labels[slug]}</span>
            </Link>
          ))}
        </nav>
      )}
    </div>
  )
}
```
